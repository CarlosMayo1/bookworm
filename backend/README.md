# A free API library

I am a book lover, I love reading many books of different genres but as many others I have the same problem, after some time I can't even remember which books I already read or how many I have read 🥺 .

That is why I decided to make a free library API, something simliar to {JSON}Placeholder. The main idea is to use real data instead of mock data but not only that, I want to make an app where people can keep a register of the books they are reading, the books they already read and those they want to read 🤓 .

## What's next?

I know it is boring right now, so I am currently working on a landing page using React as the main framework, for database I will use Supabase and for the API deployment I will use Render, so people can see it and most importantly, test it.
