# Free API version of a library
This project is going to be a free API version of a library that people can use for their own practice. The main idea is to have a place where there are many books something similar to {JSON} Placeholder(https://jsonplaceholder.typicode.com/), but instead of dummy data, it can contain real data of books.

## Context
I love reading books, my main idea to have an API of the books I read in order to have them organize and keep a track of them. For the future, I plan to make an app where people can register the books the want to read, the books they already read and those they want to read soon. This is just the first step

## What's next?
This is pretty basic, so my next step is to deploy it with an landing page so the user can see it!
